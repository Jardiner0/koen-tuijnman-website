# Koen Tuijnman - Portfolio Website

## Project info

Personal portfolio website showcasing my work as a Principal Product Manager.

## How to run this project

This project requires Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies used

- TypeScript
- React
